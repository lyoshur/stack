### 安装中文输入法

#### 安装步骤

1.安装fctix 框架

sudo apt install fctix

2.安装googlepinyin

sudo apt install fcitx-googlepinyin

3.修复依赖关系

sudo apt-get install -f

4.使用fctix configuration配置输入法
